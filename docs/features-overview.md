# DeepResearch 特性概览

## 🚀 核心特性

### 🤖 多模型 LLM 支持
- **OpenAI GPT**: GPT-4, GPT-3.5-turbo 等模型
- **Anthropic Claude**: Claude-3.5-sonnet, Claude-3-haiku 等
- **Google Gemini**: Gemini-1.5-pro, Gemini-1.0-pro 等
- **Ollama**: 本地部署的开源模型支持
- **智能模型选择**: 根据任务类型自动选择最适合的模型

### 🔍 智能搜索集成
- **多搜索引擎支持**: Google、Bing、DuckDuckGo、SerpAPI
- **智能搜索策略**: 自动选择最佳搜索引擎和查询策略

- **搜索结果缓存**: 提高搜索效率，减少 API 调用

- **代码执行**: 安全的代码执行环境（Docker/Local Sandbox）

- **文件操作**: 本地文件、Google Drive、Dropbox 集成
- **数据分析**: 统计分析、内容分析、趋势分析、情感分析
- **工具链编排**: 复杂任务的工具组合和流程编排

### 📊 结构化输出
- **Markdown 格式**: 专业的研究报告格式
- **自动目录生成**: 智能的章节结构和导航

- **多语言支持**: 支持中文、英文等多种语言输出
- **模板系统**: 可自定义的输出模板

### 🤝 交互式研究
- **用户偏好收集**: 研究深度、输出风格、语言偏好等
- **提纲确认机制**: 用户可审核和修改研究提纲
